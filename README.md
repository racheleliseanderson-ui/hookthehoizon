# Hook the Horizon

**Domain:** https://hookthehorizon.blog

Fishing, gear, and field stories from the water ahead.

This monorepo contains the independently deployable WordPress block theme and durable content plugin for Hook the Horizon. The repository slug is retained exactly as supplied; public brand text must always spell **Horizon** correctly.

## Structure

- `theme/` — presentation, block templates, tokens, patterns, responsive and accessible front-end behavior
- `content-plugin/` — durable post types, taxonomies, structured metadata, privacy controls, and shared-core integration boundary
- `docs/` — design, editorial, governance, and technical authorities
