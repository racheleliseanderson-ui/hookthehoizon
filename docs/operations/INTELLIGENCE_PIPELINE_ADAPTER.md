# Honey Hole Intelligence Pipeline Adapter

This project inherits the shared intelligence pipeline from `racheleliseanderson-ui/intelligence-core`.

Honey Hole Intelligence owns broad water, access, regulation, condition, conservation, equipment compatibility, and field-decision intelligence. Hook the Horizon owns public editorial voice and presentation.

Required controls:

- Separate sourced facts, user-provided conditions, deterministic rules, editorial judgment, and unresolved uncertainty.
- Reject exact protected locations, private access details, and sensitive coordinates from public output.
- Preserve explicit unknown, stale-source, regulation-review, access-review, and conservation-review states.
- Do not promise outcomes or represent researched information as firsthand field experience.
- Create a run manifest for every material research, modeling, correction, preview, or maintenance run.
- Preserve correction and supersession history.
- Pipeline completion does not authorize WordPress publication or production activation.

Validation must include coordinate rejection, stale regulation data, uncertain access, equipment mismatch, incomplete evidence, negative outcomes, and public-output sanitization.
