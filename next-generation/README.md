# Hook the Horizon — Next-Generation Rebuild

Status: active clean-rebuild foundation; not approved for production.

This directory starts a fresh implementation boundary for Hook the Horizon. Existing source remains forensic and rollback evidence. Strong field logic, compatibility data, venue or location intelligence, schemas, fixtures, regulation boundaries, accessibility controls, and useful editorial work may be recovered; the current visually restrained presentation is not the new foundation.

## Experience thesis

Hook the Horizon is brilliant field intelligence: open water, changing weather, reflected light, equipment relationships, route judgment, condition layers, and evidence earned outside. It should feel expansive, vivid, technically useful, and genuinely observant. It must not resemble a generic outdoor theme, tackle catalog, youth hobby site, blue-wave template, rustic man-cave brand, tournament graphic package, or faux black-and-gold luxury surface.
